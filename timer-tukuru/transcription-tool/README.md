# シンプル文字起こしツール (Transcription Tool)

このツールは、OpenAIの「Whisper」モデルを使用して、パソコン上で無料で音声ファイルの文字起こしを行います。
インターネットに音声データを送信しないため、プライバシーの面でも安心です。

## 必要なもの
1.  **Python** (インストール済みであることを確認してください)
2.  **FFmpeg** (音声処理ソフト。これがないと動きません)

### FFmpeg のインストール方法 (Windows)
1.  [FFmpeg公式サイト](https://ffmpeg.org/download.html) または [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) から Windows用のビルドをダウンロードします（`ffmpeg-release-essentials.zip` など）。
2.  ダウンロードしたZIPファイルを解凍します。
3.  解凍したフォルダの中にある `bin` フォルダ（`ffmpeg.exe` が入っている場所）のパスをコピーします。
4.  Windowsの検索バーで「システム環境変数の編集」を検索し、開きます。
5.  「環境変数」ボタンをクリックします。
6.  「システム環境変数」の `Path` を選択し、「編集」をクリックします。
7.  「新規」をクリックし、先ほどコピーしたパスを貼り付けます。
8.  「OK」を押して画面を閉じます。
9.  コマンドプロンプトを開き直し、`ffmpeg -version` と入力してバージョンが表示されれば成功です。

## インストール方法

このフォルダ（`transcription-tool`）の中で、コマンドプロンプトまたはターミナルを開き、以下のコマンドを実行してください。

```bash
pip install -r requirements.txt
```

※ 初回は時間がかかる場合があります（WhisperモデルやPyTorchなどの大きなファイルをダウンロードするため）。

## 使い方

1.  以下のコマンドでアプリを起動します。

```bash
python transcription_app.py
```

2.  「ファイルを選択」ボタンを押し、文字起こししたい音声ファイル（mp3, wav, m4aなど）を選びます。
3.  **モデル選択**: 精度を選べます。
    *   **base (標準)**: 早いですが、精度はそこそこです。
    *   **small (高精度)**: *おすすめ*。バランスが良いです。
    *   **medium (超高精度)**: かなり正確ですが、処理に時間がかかります。
    *   **large (最高精度・重い)**: 最も正確ですが、非常に時間がかかります。PCのスペックが必要です。
4.  **キーワード / ヒント (任意)**:
    *   ここに「セルフ虐待, 波風, 静かに」のように、**間違われやすい言葉**を入力しておくと、AIがそれを優先して使ってくれます。**精度向上に一番効果的です**。
5.  初回実行時はAIモデルのダウンロードが自動で行われます。
6.  文字起こしが完了すると、画面にテキストが表示されます。
5.  「テキストを保存」ボタンで結果をファイルに保存できます。
